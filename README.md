# python-ddd
Domain-Driven Design Python Application boilerplate.

## Includes

* Python 3.11
* Poetry
* `black` for formatting your code.
* `flake8` for style and quality checks.
* `mypy` for static typying.
* `pre-commit` for glueing all together.

## File structure

```pre
  ── src
      ├── application
      │   └── __init__.py
      ├── domain
      │   ├── __init__.py
      │   ├── entities.py
      │   ├── repositories.py
      │   └── value_objects.py
      └── infrastructure
          └── __init__.py
```
